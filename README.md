# Kosli Reporter

This CLI is used to report DevOps change events to Kosli and query them.

## Status
Kosli is still in beta

## Usage 

See the [docs](https://docs.kosli.com/client_reference/)

## Linting the code

`make lint`

## Building the code

`make build`

## Building the documentation

`make hugo`
